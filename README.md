## Setup

1. Clone Repository

    ```sh
    $ git clone https://github.com/San-Zero/hello_react_dapp.git
    $ cd hello_react_dapp
    ```
2. Run Dapp

    ```sh
    $ cd client
    $ npm install
    $ npm start
    ```
